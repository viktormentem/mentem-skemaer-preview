Klient-360 Fase 2: klient-profilen viser nu et forløbsdiagram pr. skala. Det nye er målestøj-båndet — tærskel-bredden er tegnet ind, så et bredt bånd læses som ærlighed om skalaens præcision, ikke som “ingen fremgang”. Og WHO-5 er af-gated: kildetallet er fundet.

Mentem → Statistik → klik på klient → sektionen “ROM-grafer”

01Sådan ser graferne ud

Gråt bånd = baseline ± Jacobson-Truax-tærsklen (alt indenfor kan ikke skelnes fra målestøj). Grøn trekant = reliabel forbedring · rust trekant = reliabel forværring — altid målt mod baseline (J-T pre→post-standard). Stiplet linje = klinisk cutoff. WHO-5-grafen viser loft-dæmpningen.

02WHO-5 af-gated — Schougaard 2018

WHO-5AF-GATED 10/6tærskel ≥6 point (rå)

Primærkilden er fundet og verificeret mod fuldteksten: Schougaard m.fl. 2018 (HQLO 16:175) — dansk WHO-5, det første test-retest-studie af skalaen overhovedet, med Per Bech (WHO-5’s ophavsmand) som medforfatter. Table 3 pooled (N=540): ICC 0.81 · SD 19.5 på 0–100-skalaen. Mentem gemmer WHO-5 som rå sum 0–25, så registry’et bærer SD = 19.5/4 = 4.875 → tærskel 5.89 rå ≡ 23.56 percent ≈ artiklens publicerede MDC95 på 23.60 (motoren reproducerer den peer-reviewede tærskel).

Loft-effekt (artiklens eget forbehold): nær toppen af skalaen måler WHO-5 mindre præcist. Derfor dæmpes RC-markeringer hvor et endepunkt ligger over max − tærskel (>19.1 af 25), med noten “nær loft”.

03Paritet JS ↔ Swift + heltals-beslutningen

Tabellen beregnes live i browseren med samme formel som Swift-motoren. UI viser heltallet ceil(tærskel) — for heltalsscores er det matematisk ækvivalent med den kontinuerte grænse og aldrig over-påstående; klassifikationen kører altid på den kontinuerte værdi.
